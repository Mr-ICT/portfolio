# Tanyaradzwa Manjengwa — Portfolio

Personal developer portfolio built with Next.js and Tailwind CSS.

## Features
- Mouse-reactive constellation canvas background
- Custom lagging cursor with magnetic hover states
- Scroll-reveal animations throughout
- Animated stats counters and skill bars
- Design Vault — a light-mode gallery showcasing graphic design work

## Stack
Next.js · TypeScript · Tailwind CSS · Framer Motion

## Run locally
```
npm install
npm run dev
```
